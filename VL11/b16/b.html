HƯỚNG DẪN GIẢI CHI TIẾT MỘT SỐ CÂU QUAN TRỌNG

Câu 5:
Đáp án A: lực hút, độ lớn 1,2×10⁻⁴ N
F = k|q₁q₂|/r² = 9×10⁹ × |2×10⁻⁹ × (-6)×10⁻⁹| / (0,03)²
= 9×10⁹ × 12×10⁻¹⁸ / 9×10⁻⁴ = 1,2×10⁻⁴ N
Hai điện tích trái dấu nên hút nhau.
Câu 7:
Đáp án B: 8,43×10⁻⁹ C
F = kq²/r² → q = √(Fr²/k)
q = √[(1,6×10⁻⁴ × (0,02)²) / (9×10⁹)]
q = √[(1,6×10⁻⁴ × 4×10⁻⁴) / (9×10⁹)]
q = √(6,4×10⁻⁸ / 9×10⁹) ≈ 8,43×10⁻⁹ C
Câu 11:
Đáp án A: -3
Ban đầu: F = k|q₁q₂|/r²
Sau tiếp xúc: q₁' = q₂' = (q₁ + q₂)/2
F' = k[(q₁ + q₂)/2]²/r² = F/3
→ 3(q₁ + q₂)² = 4|q₁q₂| → q₁/q₂ = -3
Câu 14:
Đáp án C: 16×10⁻⁴ N
OA = OB = 3 cm = 0,03 m
F₁ = k|q₁q₀|/OA² = 9×10⁹ × |4×10⁻⁸ × 2×10⁻⁹| / (0,03)² = 8×10⁻⁴ N
F₂ = k|q₂q₀|/OB² = 9×10⁹ × |-4×10⁻⁸ × 2×10⁻⁹| / (0,03)² = 8×10⁻⁴ N
F = F₁ + F₂ = 16×10⁻⁴ N (cùng chiều)
Câu 25:
Đáp án A: tăng 16 lần
F' = k|(2q₁)(2q₂)|/(r/2)² = k × 4|q₁q₂| / (r²/4)
= 4 × 4 × k|q₁q₂|/r² = 16F
Câu 33:
Đáp án B: 1,5×10⁻⁶ C
tanα = F/P = (kq²/r²) / (mg)
sinα = (a/2)/l = 3/30 = 0,1 → α ≈ 5,74°
tanα ≈ 0,1 → kq²/r² = 0,1mg
q = √[(0,1mgr²)/k] ≈ 1,5×10⁻⁶ C
Câu 40:
Đáp án C: M nằm ngoài AB, cách A 10 cm
E₁ = E₂ → k|q₁|/x² = k|q₂|/(x+10)²
2/x² = 8/(x+10)² → (x+10)² = 4x²
x+10 = 2x → x = 10 cm

CÔNG THỨC CẦN NHỚ

1. Định luật Cu-lông: F = k|q₁q₂|/(εr²)
2. Hằng số: k = 9×10⁹ N.m²/C²
3. Nguyên lý chồng chất: F = F₁ + F₂ + ...
4. Chuyển đổi: 1 nC = 10⁻⁹ C, 1 μC = 10⁻⁶ C, 1 cm = 0,01 m
